# SpoonFeed

to run backend: <br />
cd server <br />
npm install <br />
npm install multer <br />
npm install express <br />
npm install dotenv <br />
npm install cookie-parser <br />
node server.js <br />

to run frontend: <br />
cd /client/src/ <br />
npm install <br />
cd ../ <br />
cd src <br />
npm start <br />

to run mongodb: <br />
mongosh <br />
use spoonfeed; <br />
show collections; <br />

alternative if it has proper installs: <br />
choco install make (as administrator) <br />
make <br />
